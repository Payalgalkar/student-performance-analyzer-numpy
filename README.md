# Student Performance Analyzer using NumPy

## Project Description
A beginner NumPy project that analyzes student marks.

## Features
-Takes markes as input
- Total Students
- Highest Marks
- Lowest Marks
- Average Marks
- Pass/Fail Analysis
- Grade Analysis
- Topper Detection
- Students Above Average
- Rank List

## Technologies Used
- Python
- NumPy
